Project (Virtualization)

This is a virtualization project we did for our first semester. This project required us to create a virtualized network infrastructure using VSphere software that is able to host migration of Virtual Machines without any downtime. We used Distributed Resource Scheduler (DRS) together with High Availablity (HA) to achieve this.

DRS provides load balancing by migrating VMs from a heavily loaded ESXi host to another host that has enough computing resources, all while the VMs are still running. HA provides the VMs the ability to migrate to another ESXi without the need of shutting down the VMs.

This approach helps to prevent the overloading of ESXi host while overall reducing downtime. Hence this makes migration process seamless and can even be automatic.
